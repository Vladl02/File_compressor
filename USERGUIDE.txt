Set this to you compiler
    Visual Studio compiler: /utf-8 /DUNICODE
    MinGW Compiler: -finput-charset=utf-8 -DUNICODE

compress.cpp - compresses the file into a binary file
decompress.cpp - decompresses the file into a text file

file.txt - the file that is going to be compressed
fileBin.bin - the compressed version of the file
decompresedFile.txt - the decompressed version of the file

#Note pureBinarytext.cpp helps visualize the binary compression of the file
#Note verify.cpp helps verify the output with the input
